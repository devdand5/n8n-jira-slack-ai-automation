# n8n Jira Slack AI Automation

An AI-powered workflow built with n8n that monitors Jira issue updates and comments, summarizes them using OpenAI, and sends real-time notifications to Slack.

## Features

- Jira Trigger
- Event Filtering
- AI-Powered Summarization
- Slack Notifications
- Comment Monitoring
- Issue Update Monitoring

## Workflow

Jira Trigger
→ Event Filter
→ Data Processing
→ AI Agent
→ Slack Notification

## Tech Stack

- n8n
- Jira Cloud
- Slack API
- OpenAI GPT-4.1-mini

## Use Case

This workflow helps teams stay updated on Jira activity without constantly checking Jira dashboards.
